# SauceDemoFramework

A Selenium Java Automation Framework built for practicing UI automation and test framework design.

## Technologies Used

- Java
- Selenium WebDriver
- Maven
- TestNG
- Git & GitHub
- Page Object Model (POM)

## Project Structure

```
src
 └── test
     └── java
         ├── base
         ├── pages
         ├── tests
         └── utilities

testdata
pom.xml
```

## Features

- Login automation
- Data-driven testing
- External configuration using Properties files
- Page Object Model
- TestNG Data Providers
